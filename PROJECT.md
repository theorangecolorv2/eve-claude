# EVE Online Bot Research Project

## О проекте

Это **легальный исследовательский проект** по разработке бота для EVE Online с целью помочь разработчикам игры (CCP Games) улучшить свою анти-бот систему.

---

## Зачем этот проект?

### Проблема

EVE Online страдает от ботоводства, которое негативно влияет на игровую экономику и опыт игроков. Текущая анти-бот система CCP справляется недостаточно эффективно.

### Решение

Создать **функционирующего бота** с использованием реальных методов ботоводов, чтобы:
1. Продемонстрировать уязвимости существующей защиты
2. Показать разработчикам **конкретные методы**, которые используют боты
3. Помочь CCP разработать эффективные контрмеры
4. Улучшить детектирование ботов в игре

### Открытость

Весь проект **полностью публичный** (GitHub). Код, методы, техники - всё доступно разработчикам CCP для анализа.

## Цели проекта

### Основная цель

Помочь CCP Games улучшить анти-бот защиту путем демонстрации реальных методов ботоводства.

### Конкретные задачи

1. **Разработать функционирующего бота** для PvE активностей
2. **Реализовать методы хуманизации**:
   - Плавные движения мыши
   - Рандомизация таймингов
   - Естественные паттерны поведения
3. **Документировать все методы** в публичном репозитории
4. **Продемонстрировать уязвимости** текущей защиты
5. **Предоставить CCP инструменты** для улучшения детектирования

### Не-цели

Этот проект **НЕ предназначен** для:
- ❌ Реального ботоводства на live сервере
- ❌ Нарушения EULA
- ❌ Коммерческой эксплуатации
- ❌ Вреда игровому сообществу

## Технический подход

### Методы автоматизации

**Template matching** (OpenCV):
- Поиск UI элементов по скриншотам
- Простой и надежный метод
- Легко детектируется - что и нужно для исследования

**Визуальная автоматизация**:
- Бот "видит" экран как человек
- Находит элементы по изображениям
- Эмулирует мышь и клавиатуру

### Методы хуманизации

1. **Плавные движения мыши**
   - Используем кривые Безье
   - Естественная скорость и ускорение
   - Нет телепортации курсора

2. **Рандомизация таймингов**
   - Случайные задержки между действиями
   - Вариативность времени реакции
   - Избегание фиксированных паттернов

3. **Естественное поведение**
   - Ошибки и "промахи" (опционально)
   - Вариативность траекторий
   - Паузы и "размышления"

## Архитектура

### Фреймворк `eve`

Базовая библиотека автоматизации:
- `screen.py` - Захват экрана
- `vision.py` - Поиск изображений (template matching)
- `mouse.py` - Управление мышью (с хуманизацией)
- `keyboard.py` - Управление клавиатурой
- `window.py` - Управление окнами

### Скрипты автоматизации

Отдельные скрипты для различных активностей:
- Mining (майнинг астероидов)
- Ratting (фарм NPC)
- Missions (выполнение миссий)
- etc.

Каждый скрипт:
- Полностью автономный
- С подробным логированием
- С обработкой ошибок
- С методами хуманизации

## Процесс разработки

### Workflow

1. **Пользователь** определяет задачу (например, "автоматизировать майнинг")
2. **Пользователь** делает скриншоты UI элементов → `inbox/`
3. **Claude Code** обрабатывает скриншоты → `assets/`
4. **Claude Code** пишет скрипт автоматизации → `scripts/`
5. **Тестирование** на тестовом сервере
6. **Итерация** - улучшение и отладка

### Инструменты

- **Claude Code** - AI ассистент для разработки
- **Python** - язык программирования
- **OpenCV** - библиотека компьютерного зрения
- **PyAutoGUI** - эмуляция мыши/клавиатуры (с модификациями)
