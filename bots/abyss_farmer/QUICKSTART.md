# Быстрый старт Abyss Farmer

## Подготовка (5 минут)

### 1. Создайте букмарки

В космосе создайте два букмарка с точными названиями:
- `1 SPOT 1`
- `2 SPOT 2`

Это должны быть безопасные точки для варпа между забегами.

### 2. Настройте вкладки Overview

Убедитесь что у вас есть вкладки:
- `Main` - показывает контейнеры и ворота
- `PvP Foe` - показывает только врагов

Подробнее: [SETUP_TABS.md](SETUP_TABS.md)

### 3. Подготовьте корабль

- Слот 1: Ракеты (верхний слот)
- Мид слоты: Активируйте все нужные модули
- Дроны: Загрузите в дрон-бей

### 4. Подготовьте инвентарь

- Положите филаменты в инвентарь
- Настройте фильтр `!FILAMENT!` (должен показывать только филаменты)

## Запуск (1 минута)

### 1. Запустите EVE Online и Sanderling

Убедитесь что Sanderling подключен к игре.

### 2. Переместите корабль на один из спотов

Варпните на `1 SPOT 1` или `2 SPOT 2` вручную.

### 3. Запустите бота

```bash
python scripts/run_abyss_farmer.py
```

### 4. Выберите филамент

При запуске выберите тип филамента или нажмите Enter для Calm Exotic.

### 5. Бот начнет работу

Бот автоматически:
- Варпнет на спот (если нужно)
- Использует филамент
- Зачистит абисс
- Выйдет и варпнет на другой спот
- Повторит цикл

## Остановка

Нажмите `Ctrl+C` для корректной остановки.

Бот выведет статистику:
- Количество циклов
- Количество комнат
- Время работы
- Среднее время цикла

## Что делать если что-то пошло не так

### Бот не может найти букмарку

Проверьте что букмарки названы точно:
- `1 SPOT 1` (с пробелами)
- `2 SPOT 2` (с пробелами)

### Бот не может использовать филамент

Проверьте:
- Филаменты есть в инвентаре
- Фильтр `!FILAMENT!` настроен
- Инвентарь открыт или может быть открыт

### Бот не может найти врагов

Проверьте:
- Вкладка `PvP Foe` существует
- Вкладка показывает только врагов (без контейнеров и ворот)

### Бот не может найти контейнер

Проверьте:
- Вкладка `Main` существует
- Вкладка показывает контейнеры и ворота

## Логи

Все действия логируются в `logs/abyss_farmer_YYYYMMDD_HHMMSS.log`

Если что-то не работает - проверьте лог файл.

## Безопасность

- Бот чередует споты для снижения риска
- Используйте случайные задержки (уже встроены)
- Не оставляйте бота без присмотра надолго
- Проверяйте логи на ошибки
