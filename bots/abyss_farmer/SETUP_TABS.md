# Настройка вкладок Overview для Abyss Farmer

## Проблема

Бот ищет вкладки "Main" и "PvP Foe", но у тебя могут быть другие названия.

## Решение 1: Создать нужные вкладки

1. **Создай вкладку "Main":**
   - В игре открой Overview
   - Нажми на "+" (добавить вкладку)
   - Назови её "Main"
   - Настрой фильтры (показывать контейнеры, гейты, остовы)

2. **Проверь вкладку "PvP Foe":**
   - Должна показывать только врагов
   - Название должно содержать "PvP Foe"

## Решение 2: Использовать существующие вкладки

Если у тебя уже есть подходящие вкладки, просто переименуй их:

### Текущие вкладки (из лога):
```
1. '✈ Jump' - для прыжков
2. '✈ Dock' - для доков
3. '⌘ PvX +Friends' - PvX с друзьями
4. '⌘ PvX +Friends Extended' - расширенная
5. '<color=red>✥ PvP Foe</color>' - враги (ПОДХОДИТ!)
6. '❆ Resources' - ресурсы
7. 'DefaultPreset_639448' - дефолтная
8. 'DefaultPreset_639443' - дефолтная
```

### Рекомендации:

**Для вкладки "Main":**
- Используй `'DefaultPreset_639448'` или `'DefaultPreset_639443'`
- Переименуй в "Main" или добавь "Main" в название
- Настрой фильтры: контейнеры, гейты, остовы

**Для вкладки "PvP Foe":**
- У тебя уже есть `'<color=red>✥ PvP Foe</color>'` - ОТЛИЧНО!
- Бот найдёт её автоматически (игнорирует HTML теги)

## Решение 3: Изменить код

Если не хочешь менять вкладки, измени код в `room_new.py`:

```python
# Вместо:
if not _switch_to_tab(sanderling, "Main"):

# Используй:
if not _switch_to_tab(sanderling, "DefaultPreset"):  # Найдёт любую DefaultPreset
```

## Проверка настроек

Запусти скрипт для проверки вкладок:

```bash
python scripts/check_overview_tabs.py
```

Он покажет все доступные вкладки и их названия.

## Как работает поиск вкладок

Бот ищет вкладки по **частичному совпадению**, игнорируя:
- HTML теги (`<color=red>`, `</color>`)
- Спецсимволы (✈, ⌘, ✥, ❆)
- Регистр букв

Примеры:
- Запрос "PvP Foe" найдёт `'<color=red>✥ PvP Foe</color>'`
- Запрос "Main" найдёт `'Main'` или `'⌘ Main Overview'`
- Запрос "DefaultPreset" найдёт `'DefaultPreset_639448'`

## Рекомендуемая настройка

1. Переименуй `'DefaultPreset_639448'` в `'Main'`
2. Оставь `'<color=red>✥ PvP Foe</color>'` как есть
3. Настрой фильтры:
   - **Main:** контейнеры, гейты, остовы, всё что нужно видеть
   - **PvP Foe:** только враги (NPC)
