# Использование Abyss Farmer - Default Room

## Быстрый старт

1. **Подготовка корабля**
   - Настройте хоткеи:
     - `1` - ракеты (верхний слот)
     - `2` - МВД (средний слот, левый)
     - `3` - второй мид слот (правый)
     - `Shift+F` - выпуск дронов
     - `F` - атака дронами

2. **Настройка overview**
   - Создайте вкладку `Main` - основная вкладка
   - Создайте вкладку `PvP Foe` - только враги

3. **Запуск Sanderling**
   ```bash
   # Запустите Sanderling.exe
   # Подключитесь к процессу EVE Online
   ```

4. **Запуск бота**
   ```bash
   python scripts/test_default_room.py
   ```

## Что делает бот

### Фаза 1: Подготовка (Main)
- Переключается на вкладку Main
- Проверяет мид слоты
- Ждет контейнер
- Выходит на орбиту контейнера
- Выпускает дронов

### Фаза 2: Первая цель (PvP Foe)
- Переключается на PvP Foe
- Ждет цель в радиусе 35км
- Лочит и убивает первую цель

### Фаза 3: Контейнер (Main)
- Переключается на Main
- Лочит контейнер
- Убивает ракетами
- Лутает врек

### Фаза 4: Гейт (Main)
- Находит Acceleration Gate
- Выходит на орбиту

### Фаза 5: Зачистка (PvP Foe)
- Переключается на PvP Foe
- Убивает всех врагов волнами

## Логи

Логи сохраняются в:
```
logs/test_default_room_YYYYMMDD_HHMMSS.log
```

## Отладка

Если что-то пошло не так:

1. **Проверьте логи** - там подробная информация о каждом шаге
2. **Проверьте вкладки overview** - должны быть Main и PvP Foe
3. **Проверьте хоткеи** - должны совпадать с настройками
4. **Проверьте Sanderling** - должен быть подключен к игре

## Известные проблемы

- Если контейнер не появляется - проверьте что вы в комнате Абисса
- Если цели не лочатся - проверьте что вы на правильной вкладке
- Если дроны не выпускаются - проверьте хоткей Shift+F
