✅ ТЕПЕРЬ ЗАПУСКАЕТСЯ ОДНОЙ КОМАНДОЙ!

┌──────────────────────────────────────────────────────────────┐
│  1. Установить библиотеки:                                   │
│     pip install python-telegram-bot python-dotenv            │
│                                                              │
│  2. Создать .env файл:                                       │
│     cp .env.example .env                                     │
│     Добавить токен: TELEGRAM_BOT_TOKEN=ваш_токен             │
│                                                              │
│  3. Запустить бота:                                          │
│     python scripts/eve_farm_bot.py                           │
│                                                              │
│     ► Telegram бот запустится автоматически в фоне!          │
│                                                              │
│  4. Написать боту в Telegram:                                │
│     /start                                                   │
└──────────────────────────────────────────────────────────────┘

═══════════════════════════════════════════════════════════════

📢 ЧТО ИЗМЕНИЛОСЬ:

  ✅ Telegram бот запускается АВТОМАТИЧЕСКИ
  ✅ Не нужно запускать 2 терминала
  ✅ Ctrl+C останавливает оба бота
  ✅ Если python-telegram-bot не установлен - бот работает БЕЗ уведомлений

═══════════════════════════════════════════════════════════════

📋 ФАЙЛЫ:

  .env                         - ТОКЕН БОТА (создай из .env.example)
  .env.example                 - Шаблон для .env
  scripts/eve_farm_bot.py      - ГЛАВНЫЙ ФАЙЛ (запускаешь его)
  scripts/telegram_bot.py      - Устарел (можно удалить)
  eve/telegram_notifier.py     - Модуль уведомлений
  data/telegram_users.json     - Подписчики (создастся автоматически)
  data/bot_stats.json          - Статистика (создастся автоматически)

═══════════════════════════════════════════════════════════════

📱 КОМАНДЫ БОТА:

  /start  - Подписаться + статистика
  /stats  - Показать статистику
  /users  - Количество подписчиков
